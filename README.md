# ⚡ zapret-roblox-youtube

Сборка утилиты `zapret` для автоматизированного обхода замедлений и блокировок популярных сервисов: **Roblox, Discord, YouTube** и других ресурсов. Полностью готовое решение для Windows, не требующее ручной настройки конфигурационных файлов.

---

## 🚀 Быстрый старт

1. **Скачайте архив** последней версии со страницы [Releases](https://github.com/maneetoo/zapret-roblox-youtube/releases/latest).
2. **Снимите блокировку системы** (Windows часто блокирует исполняемые файлы из интернета):
   * Кликните правой кнопкой мыши по скачанному архиву `->` **Свойства**.
   * На вкладке *Общие* (внизу) поставьте галочку **Разблокировать** и нажмите *Применить*.
3. **Распакуйте архив** в любую удобную папку. 
   > ⚠️ **Важно:** Путь к папке не должен содержать пробелов и кириллицы (например, `C:\zapret-roblox`).
4. **Запустите** файл `service.bat` строго от имени **Администратора**.
5. **Установите службу**:
   * В открывшемся консольном меню выберите опцию `Install Service`.
   * Выберите желаемую стратегию обхода (для базовой настройки рекомендуется начать с **ALT** или **FAKE**).

---

## 🛠️ Что делать, если не работает?

Если после установки службы блокировки не снялись, попробуйте следующие шаги:

* **Смените стратегию:** Снова запустите `service.bat` от имени администратора, выберите удаление старой службы и установите новую с другим пресетом (например, переключитесь на альтернативный режим DPI-bypass).
* **Включите Secure DNS (DoH):** * В браузере (Chrome/Edge/Opera): *Настройки -> Конфиденциальность и безопасность -> Использовать безопасный DNS-сервер* (выберите Cloudflare `1.1.1.1` или Google `8.8.8.8`).
* **Очистите кэш DNS:** Откройте командную строку (`cmd`) от администратора и выполните команду:

```bash
ipconfig /flushdns
```

* **Добавьте свои домены:** Если какой-то конкретный сайт или игра всё ещё недоступны, откройте текстовый файл `list-general-user.txt` в папке проекта и внесите туда нужные доменные имена (каждое с новой строки), после чего перезапустите службу.
